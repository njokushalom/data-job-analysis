# 📊 Project on Data Job Analysis

Hello everyone,

In this project, I explore the main objective of analyzing job trends within the data science space using Excel add-ins, specifically **What-If Analysis tools**.

---

## 🔹 Scenario Manager

Let’s consider a real-life situation.

You have multiple job offers but feel uncertain about which one to choose due to differences in salary structure, bonuses, and long-term growth. In reality, many roles today may offer modest base salaries, while total compensation is often influenced heavily by bonuses and annual raises.

This is where **Scenario Manager** becomes a powerful tool.

In this analysis, I evaluated **10 data job offers**, each with a base salary of **$123,129**. However, each role varies in:

- Bonus percentages  
- Annual raise rates  

Using Scenario Manager, I created multiple compensation scenarios that automatically calculate total earnings based on these variables.

![Scenario Manager](scenario.png)

---

## 🔹 Taking It Further: Solver

Now, let’s take it a step further.

Suppose you want to negotiate an offer without changing the base salary. This is where **Solver** comes in.

Solver helps determine the optimal values needed to achieve a desired outcome.

In this case:

- Original values: **5% bonus** and **10% annual raise**  
- Solver adjusted these to reach a target salary  

It answers the question:  
**“What bonus and raise structure do I need?”**

![Solver](solver.png)

---

## 🔹 Key Insight

With Solver, both the bonus and annual raise are automatically adjusted to meet a defined salary goal. This makes salary negotiation more data-driven and strategic.

This project demonstrates how Excel tools like Scenario Manager and Solver can transform decision-making.
